# HTML 작성 순서 
------------------
1. 구역을  semantic tag로 나눕니다.
> html 을 가로+세로로 영역을 먼저 나눕니다.

2. 나눈 구역에 보이는 컨텐츠들을 채워넣습니다.
> ex) Cities , London.. 이런 내용들을 채워넣습니다.

3. 헤더 - 메인 - 푸터 순서(위에서 아래)로 css를 채워넣습니다.
> css를 적용할때, 전체적인 밸런스를 고려해서 작성합니다.

4. 원본과 비교하며 적용합니다.
